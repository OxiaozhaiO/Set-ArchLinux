# 錯誤修復
### 1.FILESYSTEM CHECKED FAILED
```java
fsck [磁碟路徑]
比如:fsck /dev/sda3
```
### 2./dev/dsp not found
```java
sudo mknod /dev/dsp c 14 3  
sudo chmod 666 /dev/dsp 
```
     如果上面指令不起作用,再執行下面的代碼
```java
sudo modprobe  snd-pcm-oss  
sudo modprobe  snd-mixer-oss  
```
### 3.套件不正確或損毀（PGP 簽章）
```java
sudo pacman -Sy archlinux-keyring &&sudo pacman -Syyu
```
